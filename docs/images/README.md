# Images Directory

This directory contains images for documentation.

## Structure

```
images/
├── banner.png           # Main plugin banner for README
└── screenshots/
    ├── claim-creation.png
    ├── guild-panel.png
    ├── permissions.png
    └── bedrock-forms.png
```

## Recommended Sizes

- **Banner**: 1200x400px or similar aspect ratio
- **Screenshots**: 800x600px or 16:9 aspect ratio

## Adding Images

1. Add your image files to the appropriate directory
2. Uncomment the corresponding image references in README.md or docs files
